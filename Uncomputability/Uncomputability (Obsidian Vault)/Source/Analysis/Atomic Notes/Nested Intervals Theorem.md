
>[!theorem]  Лемма Коши-Kантора о вложенных отрезках
> Для любой [[Nested Sets|убывающей последовательности вложенных]] [[Interval|отрезков]] $I_1 ⊇ I_2 ⊇ \ldots$ найдется точка $c∈ℝ$, принадлежащая этим отрезкам:
> 
> $$
> с∈ \bigcap_{n=1}^∞
> $$
> 
> Если при этом последовательность является **стягивающейся**, т.е. для любого $ε>0$ найдется такое значение $N∈ℕ$, что для всех $n>N$ длины отрезков $I_n$ будет меньше $N$, то пересечение всеx $I_n$ является [[Singleton|синглетоном]]:
> 
> $$
> \bigcap_{n=1}^∞ = \{ c \}
> $$
> 

Лемма следует из [[Dedekind Completeness|принципа Дедекинда]] и для архимедовых [[Ordered Field|упорядоченных полей]] эквивалентна [[Dedekind Completeness|ему]].

>[!detail]- последовательности вложенных открытых интервалов
>Заметим, что для последовательности вложенных открытых интервалов такое утверждение будет неверным: последовательность $(0, \frac{1}{n})$ имеет пустое пересечение, так как для любого $c∈ℝ$ найдется $n$-ый открытый интервал, который не содержит $c$.
